# CCNA (Cisco Certified Network Associate) 学習記録

2023年度上半期の集大成。ネットワークの基礎から実機構成までを体系的に学習した記録。
2023年7月15日の試験合格に向け、15日前の6月末時点で全検証を完了し、本資料をFix。

## 1. ネットワーク基礎
- **OSI参照モデル / TCP/IPスタック**: 各レイヤのカプセル化・非カプセル化プロセスの詳細理解。
- **IPアドレッシング**: VLSM（可変長サブネットマスク）を用いた効率的なネットワーク設計とサブネット計算。

## 2. レイヤ2スイッチング技術
- **VLAN & Trunking**: IEEE 802.1Qカプセル化、ネイティブVLANのセキュリティ考慮事項。
- **Spanning Tree Protocol (STP)**: 
  - 802.1D（レガシーSTP）と802.1w（RSTP）の収束プロセス比較。
  - ルートブリッジ選定アルゴリズムとポート状態遷移のパケット解析。
- **EtherChannel**: LACPを用いた帯域集約と、ハッシュアルゴリズムによる負荷分散の挙動確認。

## 3. レイヤ3ルーティング技術
- **OSPF (Open Shortest Path First)**:
  - Helloパケットによるネイバー関係の確立、LSAのタイプとエリア設計（Single/Multi Area）。
  - コスト計算に基づく最短経路選定（SPFアルゴリズム）の検証。
- **スタティックルーティング**: フローティングスタティックを用いたバックアップルートの冗長化構成。

## 4. インフラストラクチャ・サービスとセキュリティ
- **ACL (Access Control List)**: 標準/拡張ACLを用いたL3/L4フィルタリングの実装。
- **NAT / PAT**: スタティックNAT、ダイナミックNAT、およびオーバーロード（NAPT）の変換テーブル管理。
- **管理プロトコル**: SNMP, Syslog, NTPを用いたネットワーク監視基盤の基礎。
